# 假加载效果（微交互）

**日期**: 2025-12-07  
**提交**: `2e6d471`

## 开发需求
搜索结果更新时缺乏视觉反馈，用户可能察觉不到内容已刷新（"Change Blindness"问题）。需要添加微交互提示。

## 实现原理
监听 `results` 数组变化，触发时设置 `isRefreshing = true`，应用 `opacity-50 blur-[2px]` 样式，300ms后恢复。通过CSS transition实现平滑的"呼吸与模糊"效果。

## 最终效果
搜索结果更新时，结果列表会短暂降低透明度并轻微模糊，然后平滑恢复，为用户提供清晰的视觉反馈，提升交互体验。
